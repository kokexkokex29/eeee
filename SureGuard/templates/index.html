Setup Instructions

1. Add Discord Token

في تبويب Secrets في Replit، أضف:

DISCORD_TOKEN = your_bot_token_here
احصل على التوكن من Discord Developer Portal
2. Deploy to Render
  1. اربط مستودع GitHub الخاص بك بـ Render
  2. اختر "Web Service"
  3. Build Command: pip install -r requirements.txt
  4. Start Command: python main.py
  5. أضف متغير البيئة DISCORD_TOKEN
نصيحة: تأكد من أن البوت لديه الصلاحيات التالية في Discord: Manage Roles, Send Messages, Use Slash Commands
